# Container Installation Scripts
Some handy scripts to install [Docker](https://www.docker.com), [Docker-Compose](https://docs.docker.com/compose/), [K3s](https://k3s.io), and [K9s](https://github.com/derailed/k9s) in Arch Linux system.

Reference:
- https://docs.docker.com/engine/install/
- https://docs.docker.com/compose/install/
- https://rancher.com/docs/k3s/latest/en/
- https://github.com/derailed/k9s
